Latest Projects

FoodMo

FoodMo is a full-stack web application built using React.js, Node.js, Express.js, and MongoDB. It offers a smooth and responsive interface for users to browse food items, add them to a cart, and place orders in real-time. The project includes features like user authentication, role-based access for admins and customers, and a dynamic admin panel for managing menu items and tracking orders. The frontend is optimized with Vite and styled using modern UI techniques to ensure fast performance and an intuitive user experience.

Home Finds

Home Finds is a modern real estate platform built with the MERN stack, designed to simplify property discovery and management. It offers an intuitive user interface, advanced search filters, and secure authentication for seamless property browsing and listing. With real-time data updates and a responsive design, Home Finds ensures a smooth experience for both buyers and sellers, making house-hunting easier than ever.

Library Management System

The Library Management System is a Java-based project that utilizes Data Structures and Algorithms (DSA) to efficiently manage books, users, and borrowing records. It allows users to add, remove, and search for books using optimized algorithms like HashMaps and Trees. The system maintains user details, tracks book issuance and returns, and ensures seamless management of due dates. With efficient searching and sorting techniques such as Binary Search and Hashing, book retrieval is fast and accurate. Data persistence is achieved through file handling or database integration, making the system scalable and practical for real-world applications. This project showcases the power of Java and DSA in solving real-world management problems effectively.
